Como criar um Fork do repositório

Passo a Passo

  1. Acesse o repositório original: 100_dias_de_codigo
  2. Clique no botão Fork no canto superior direito da página.
  3. Escolha onde você quer criar o fork (na sua conta).
  4. Aguarde enquanto o GitHub cria o fork.
  5. Após o processo, você será redirecionado para o seu novo repositório forked.
